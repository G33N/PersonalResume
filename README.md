## How install
This APP required someone dependencies they are:
1. npm
2. angular-cli
3. git


When you have ready the dependencies just follow this tasks.

You should type this:
1. `git clone https://github.com/G33N/PersonalResume.git`
2. `cd SCYS`
3. `npm install`
4. `bower install`

End.
